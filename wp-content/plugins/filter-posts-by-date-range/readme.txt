Filter your WordPress posts by date range

== Description ==
This plugin give you ability to filter your WordPress posts by date range.

Usage :

1. Download and extract this plugin to `wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to 'Posts' menu & start using this awesome plugin.

== Installation ==
1. Download and extract this plugin to `wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to 'Posts' menu & start using this awesome plugin.
